Demo Hệ thống Phân quyền QLDRL

Hướng dẫn sử dụng

1. Mở file index.html trong trình duyệt

2. Nhấn nút "Đăng nhập" để mở modal đăng nhập

3. Sử dụng các tài khoản demo bên dưới để test phân quyền

Tài khoản Admin

Tên đăng nhập: admin

Mật khẩu: admin123

Mã quản trị: AD01

Kết quả: Chuyển đến giao diện admin

Tài khoản Giảng viên

Tên đăng nhập: giangvien

Mật khẩu: gv123

Mã quản trị: GV01

Kết quả: Chuyển đến giao diện giảng viên

Tài khoản Sinh viên

Tên đăng nhập: sinhvien

Mật khẩu: sv123

Mã quản trị: SV01

Kết quả: Ở lại trang chủ

Lưu ý quan trọng

• Đây chỉ là demo giao diện. API thực tế cần được cấu hình để hoạt động đúng.

• Hệ thống sử dụng Mã quản trị (MaQT) để phân quyền:

  • • AD01: Quản trị viên → Giao diện admin
  • • GV01: Giảng viên → Giao diện giảng viên
  • • Khác: Sinh viên → Ở lại trang chủ

• Nếu không có API, bạn có thể thay đổi logic trong script.js để test.

• Mỗi giao diện (admin, giảng viên) đều có kiểm tra quyền truy cập riêng.
